# Wang_NG_Project4

This repo is for 1056 Project 4 - Responsive Web Design - Media Query created by Naigong Wang.

This repo will demonstrate the ability to Created a single web page with a basic example of a media query.


## Prerequisites
All you need is use computer browser with an Internet connection.

## Usage
Check the main branch, clone the project, cd to your directory/ download and You're good to go!

## Responsive Details
Single Web Page includs 4 Columns
At mobile size (remember mobile first)
- The columns should stack and be full width.
- Columns 3 and 4 should be hidden
- Column 1 should be pink
- Column 2 should be red
At 641px the page will have
- 4 columns, equal width and display side by side.
- Column 1 - blue, Column 2 - purple, Column 3 - grey, Column 4-brown.
- The site will not expand beyond 1140px
